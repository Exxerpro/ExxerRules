# Root Cause Analysis Checklist

## Problem Definition
- [ ] **Clear Problem Statement**
  - [ ] Issue described in specific terms
  - [ ] Symptoms clearly documented
  - [ ] Impact quantified
  - [ ] Timeline established
  - [ ] Stakeholders identified

- [ ] **Context Information**
  - [ ] System environment documented
  - [ ] Configuration details recorded
  - [ ] Recent changes identified
  - [ ] Dependencies mapped
  - [ ] User reports collected

## Data Collection
- [ ] **Evidence Gathering**
  - [ ] Logs collected and analyzed
  - [ ] Error messages documented
  - [ ] Performance metrics gathered
  - [ ] User reports reviewed
  - [ ] System state captured

- [ ] **Data Sources**
  - [ ] Application logs
  - [ ] System logs
  - [ ] Database logs
  - [ ] Network logs
  - [ ] User feedback

## Hypothesis Formation
- [ ] **Initial Hypotheses**
  - [ ] Code logic issues
  - [ ] Data integrity problems
  - [ ] Resource constraints
  - [ ] Timing issues
  - [ ] Configuration problems

- [ ] **Hypothesis Documentation**
  - [ ] Each hypothesis clearly stated
  - [ ] Supporting evidence identified
  - [ ] Test methods defined
  - [ ] Expected outcomes specified
  - [ ] Priority assigned

## Investigation Process
- [ ] **Systematic Testing**
  - [ ] Each hypothesis tested
  - [ ] Evidence collected for each
  - [ ] Results documented
  - [ ] Contradicting evidence noted
  - [ ] Confidence levels assessed

- [ ] **Data Analysis**
  - [ ] Patterns identified
  - [ ] Correlations found
  - [ ] Trends analyzed
  - [ ] Anomalies detected
  - [ ] Statistics calculated

## Root Cause Identification
- [ ] **Primary Root Cause**
  - [ ] Root cause clearly identified
  - [ ] Evidence supports conclusion
  - [ ] Confidence level high
  - [ ] Contributing factors identified
  - [ ] Causal chain documented

- [ ] **Validation**
  - [ ] Root cause verified
  - [ ] Alternative explanations ruled out
  - [ ] Evidence is conclusive
  - [ ] Expert review completed
  - [ ] Stakeholder agreement obtained

## Impact Analysis
- [ ] **Immediate Impact**
  - [ ] Business impact assessed
  - [ ] User impact quantified
  - [ ] System impact measured
  - [ ] Financial impact calculated
  - [ ] Reputation impact evaluated

- [ ] **Long-term Impact**
  - [ ] Future risks identified
  - [ ] Trend analysis completed
  - [ ] Mitigation strategies planned
  - [ ] Prevention measures designed
  - [ ] Monitoring requirements defined

## Solution Design
- [ ] **Immediate Solutions**
  - [ ] Quick fixes identified
  - [ ] Workarounds implemented
  - [ ] Temporary measures taken
  - [ ] User communication sent
  - [ ] Monitoring enhanced

- [ ] **Long-term Solutions**
  - [ ] Root cause addressed
  - [ ] Architecture improved
  - [ ] Process enhanced
  - [ ] Training provided
  - [ ] Documentation updated

## Prevention Measures
- [ ] **Process Improvements**
  - [ ] Procedures updated
  - [ ] Workflows improved
  - [ ] Standards enhanced
  - [ ] Guidelines created
  - [ ] Training materials developed

- [ ] **Technical Improvements**
  - [ ] Monitoring added
  - [ ] Alerts configured
  - [ ] Automation implemented
  - [ ] Redundancy added
  - [ ] Security enhanced

## Documentation
- [ ] **Analysis Report**
  - [ ] Problem statement clear
  - [ ] Investigation process documented
  - [ ] Root cause identified
  - [ ] Evidence presented
  - [ ] Recommendations provided

- [ ] **Knowledge Transfer**
  - [ ] Lessons learned documented
  - [ ] Best practices updated
  - [ ] Team briefed
  - [ ] Training provided
  - [ ] Process improved

## Quality Assurance
- [ ] **Analysis Quality**
  - [ ] Methodology followed
  - [ ] Evidence is sufficient
  - [ ] Conclusions are sound
  - [ ] Recommendations are actionable
  - [ ] Documentation is complete

- [ ] **Process Quality**
  - [ ] Timeline met
  - [ ] Resources used effectively
  - [ ] Communication clear
  - [ ] Stakeholders satisfied
  - [ ] Results delivered

## Success Criteria
- [ ] **Root Cause Identified**
  - [ ] Primary cause found
  - [ ] Contributing factors identified
  - [ ] Evidence is conclusive
  - [ ] Confidence level high
  - [ ] Stakeholder agreement

- [ ] **Actionable Recommendations**
  - [ ] Immediate actions defined
  - [ ] Long-term solutions planned
  - [ ] Prevention measures designed
  - [ ] Monitoring requirements specified
  - [ ] Success metrics defined
