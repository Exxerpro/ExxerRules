# Due Diligence Reports

This folder stores JSON outputs generated by:

- `src/scripts/DueDiligence-PreStart.ps1`
- `src/scripts/DueDiligence-PostFinish.ps1`

Each file is named `<WorkItemId>-prestart.json` or `<WorkItemId>-postfinish.json` and contains the checkpoints required for programme governance.
